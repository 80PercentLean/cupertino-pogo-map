# Contributing

Cupertino PoGO Map is an open source project where development happens in the open on our [GitHub repository](https://github.com/80PercentLean/cupertino-pogo-map). We welcome anyone to contribute, simply [create a pull request](https://github.com/80PercentLean/cupertino-pogo-map/pulls) and we'll review it when we can!

To see what technologies we use, read the ["Technology Overview" section in `README.md`](../README.md#technology-overview).
